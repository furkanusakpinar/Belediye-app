### 🚀 Features
- 📰 **News & Alerts**: Live municipal updates.
- 💳 **E-Services**: Online debt inquiry and payments.
- 💊 **Pharmacies**: Real-time duty pharmacy locator.
- 🏰 **City Guide**: Discovery tool for local attractions.
- 🌙 **Premium UI**: Elegant Dark Mode experience.

---

### 🛠️ Tech Stack
- **Frontend**: React Native (Expo)
- **Navigation**: Expo Router
- **Data**: NewsAPI & Firebase

---

### 💻 Quick Start
1. **Install**: `npm install`
2. **Env**: Add `EXPO_PUBLIC_NEWS_API_KEY` to `.env`
3. **Run**: `npx expo start`
